### Key findings:
- Mintlify is a docs-as-code platform: Git repo with MDX files + docs.json config → deployed docs site
- Three main surfaces: Dashboard (management), Web Editor (content editing), Docs Site (public-facing)
- Auth: Google/GitHub OAuth for dashboard, multiple methods (OAuth/JWT/password) for docs site auth
- API: Bearer token auth with mint_/mint_dsc_ prefixed keys, 16 endpoints
- AI features: Assistant (chat widget), Agent (auto-update PRs), MCP server, llms.txt
- CLI: mint dev, mint build, mint analytics, mint broken-links
- Rich component library: 25+ MDX components (Callouts, Cards, Tabs, Steps, Code Groups, etc.)
- Team management with RBAC (admin/editor/viewer)
- Analytics: page views, visitors, searches, feedback, assistant conversations
